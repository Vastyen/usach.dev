Direcciones de Red

¿Como se reconoce a un dispositivo en las redes?

La red tiene direcciones asociadas a cada dispositivo que permiten reconocer de donde a donde enviar los paquetes de datos que viajan por esta.

¿Qué tipos de direcciones hay?

Existen los siguientes tipos de direcciones:
  • Fisicas: Direcciones que vienen de por si con el dispositivo, estas son las MAC(Media Access Control) de las tarjetas de red.
    Generalmente este tipo de dirección se ocupa en transferencias de datos de bajo nivel.

  • Logicas: Direcciones que son asignadas por algun software a los dispositivos, estas son las IP(Internet Protocol) asignadas cuando uno se conecta a una red.
    Este tipo de dirección se ocupa en transferencias de datos mas complejas que las fisicas.

¿Cómo se elige cual ocupar?

Por defecto todos los dispositivos con posibilidad de conectarse a redes ocupan la dirección MAC en la comunicación inicial con está, ya que con esta dirección el Switch o Enrutador correspondiente sabe donde enviar la información de asignación de direcciones logicas.
Posteriormente, si ya se le configuro una dirección logica al dispositivo, está sera la ocupada para realizar las siguientes conexiones.
